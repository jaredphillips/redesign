MAKE YOUR OWN EMMET SHORTCUT

We got this on lock

Posted on: June 27, 2014

Tweet

It was about the 4000 time that I typed “margin: 0 auto;” that I decided it was time to make my own binding in Emmet. It turns out, it was super simple and here’s how you do it (Assuming you have Emmet installed).

Open Sublime Text. Go Preferences -> Package Settings -> Emmet -> Settings - User

This should be a blank JSON file.

After doing some research, it appears that this is the magic formula for creating your own binding.

With this info we can get started making our own. Fools think we'd be playing, but its about to get real in here.

I wanted to have two snippets, one for that dreaded 'margin: 0 auto;' and one to put some Twin Peaks Ipsum into projects.

For each file extension, you're going to have to declare the snippet. I write mostly in "scss" but just incase I have to write some straight up css, its on lock. If you get an error, JSON is a fickle beast. It doesn't care about white space but you're probably missing a comma. It helped me to think of hashes inside hashes and where those commas lined up.

Fools be typing 'margin: 0 auto' all day, I'd be all 'mz'.
